## Intro

My ESLint config for TS React projects. It uses https://www.npmjs.com/package/@rushstack/eslint-patch to bundle all necessary plugins.


## Install

```sh
npm i @wallopsio/eslint-config-tsx
```


## Use

```js
module.exports = {
  'root': true,
  'extends': [
    '@wallopsio/eslint-config-tsx',
  ]
};
```
